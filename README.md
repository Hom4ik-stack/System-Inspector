System Inspector
System Inspector — это утилита для комплексного мониторинга состояния операционной системы Windows, управления оборудованием и аудита информационной безопасности. Приложение предоставляет детальную информацию об аппаратном обеспечении, запущенных процессах, драйверах и периферийных устройствах, а также выполняет поиск уязвимостей.

Функциональные возможности
Программа разделена на несколько функциональных модулей (вкладок):

1. Информация о системе
Отображает сводные технические характеристики ПК, полученные через WMI (Windows Management Instrumentation):

Версия и сборка операционной системы.

Информация о процессоре (CPU), оперативной памяти (RAM), материнской плате и BIOS.

Список сетевых адаптеров.

Анализ актуальности версии Windows.

2. Производительность
Мониторинг ресурсов системы в реальном времени:

График загрузки центрального процессора (используются системные счетчики производительности).

Таблица запущенных процессов с указанием потребления памяти и нагрузки на ЦП.

Функция безопасного завершения процессов (встроена защита от завершения критически важных системных процессов Windows).

3. Память и Диски
Визуализация состояния файловой системы:

Список всех логических и физических дисков.

Отображение типа диска, файловой системы и свободного места.

Диаграммы заполненности накопителей.

4. Драйверы
Инструмент для аудита установленных драйверов:

Вывод списка всех подписанных драйверов в системе.

Анализ на предмет устаревания (сравнение даты драйвера с текущей датой, учет типа устройства).

Проверка наличия цифровой подписи.

Классификация рисков (Критический/Средний/Низкий).

5. Устройства
Мониторинг периферии в режиме реального времени:

Автоматическое обнаружение подключения и отключения устройств (USB, Bluetooth, PnP) без необходимости ручного обновления.

Идентификация типа устройства (Клавиатура, Мышь, Накопитель, Телефон и т.д.).

Проверка безопасности подключенных устройств (предупреждения о съемных носителях, устройствах с ошибками).

Возможность безопасного извлечения устройств и открытия их настроек.

6. Безопасность
Центр управления защитой системы:

Антивирус: Определение установленного антивируса, его статуса, версии баз и состояния защиты в реальном времени (поддержка Windows Defender и сторонних решений).

Комплексное сканирование:

Проверка критических уязвимостей (SMBv1, RDP, UAC).

Анализ открытых сетевых портов и активных соединений.

Проверка целостности файла hosts.

Анализ автозапуска и политик безопасности.

Журнал событий: Вывод последних событий безопасности Windows (неудачные попытки входа, сбои приложений, угрозы Defender).

7. Отчетность
Генерация подробного HTML-отчета о состоянии системы.

Экспорт списка угроз, характеристик и рекомендаций по безопасности.

Техническое описание
Программа написана на языке C# с использованием платформы WPF (Windows Presentation Foundation) и паттерна проектирования MVVM.

Используемые технологии:
WMI (Windows Management Instrumentation): Основной источник данных. Используется для получения информации о железе (Win32_Processor, Win32_DiskDrive), драйверах (Win32_PnPSignedDriver) и антивирусах (root\SecurityCenter2).

WMI Events: Используется для реактивного отслеживания подключения устройств (__InstanceCreationEvent), что исключает необходимость постоянного опроса системы таймером.

Windows Registry: Используется для проверки настроек безопасности (UAC, Firewall, автозагрузка).

WinAPI / P/Invoke: Используется для получения расширенной информации о сетевых TCP/UDP таблицах и процессах.

LiveCharts: Библиотека для визуализации графиков производительности.

Требования к системе
ОС: Windows 7, 8, 10, 11.

Платформа: .NET (версия зависит от сборки, рекомендуется актуальная версия .NET Desktop Runtime).

Права доступа: Для корректной работы программы (доступ к WMI, реестру и управлению процессами) требуются права Администратора.

Установка и запуск
Скачайте исполняемый файл или скомпилируйте проект из исходного кода.

Запустите файл SystemInspector.exe.

При запросе UAC предоставьте приложению права администратора.

Сборка из исходного кода
Клонируйте репозиторий.

Откройте решение .sln в Visual Studio 2022 или новее.

Выполните восстановление NuGet пакетов.

Соберите проект в конфигурации Release или Debug.
